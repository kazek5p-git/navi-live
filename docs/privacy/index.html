Polityka prywatności

O aplikacji

Navi Live to aplikacja do pieszej nawigacji, projektowana z myślą o osobach niewidomych i słabowidzących.

Z czego korzysta aplikacja

  • z lokalizacji urządzenia, aby wyszukiwać pobliskie miejsca, wyznaczać trasy piesze i prowadzić aktywną nawigację,
  • z pamięci lokalnej urządzenia, aby zapisywać ulubione miejsca, ustawienia, stan samouczka i ostatnią trasę,
  • z dostępu do sieci, aby wysyłać zapytania wyszukiwania, geokodowania wstecznego i routingu do publicznych usług opartych o OpenStreetMap.

Czego aplikacja obecnie nie używa

  • kont użytkowników,
  • SDK reklamowych,
  • SDK analitycznych,
  • SDK do raportowania awarii,
  • zakupów w aplikacji,
  • płatności,
  • czatu lub wiadomości między użytkownikami.

Jak wykorzystywana jest lokalizacja

Lokalizacja jest używana wyłącznie do:

  • ustalenia bieżącej pozycji,
  • wyszukiwania pobliskich miejsc,
  • wyznaczania i przeliczania tras,
  • prowadzenia aktywnej nawigacji.

Dane lokalne

Na urządzeniu mogą być zapisywane między innymi:

  • ulubione miejsca,
  • ustawienia aplikacji,
  • stan samouczka i onboardingu,
  • stan ostatniej trasy.

Dane te służą wyłącznie do poprawy działania aplikacji na tym samym urządzeniu.

Usługi zewnętrzne

Aplikacja wysyła niezbędne zapytania do publicznych usług wyszukiwania i routingu opartych o OpenStreetMap, aby realizować podstawowe funkcje nawigacyjne.

Zapytania te mogą obejmować:

  • tekst wyszukiwanego celu,
  • przybliżone punkty trasy,
  • współrzędne potrzebne do wyznaczenia trasy lub geokodowania wstecznego.

Udostępnianie danych

Aplikacja nie utrzymuje obecnie własnego systemu kont użytkowników ani chmury powiązanej z Twoją tożsamością.

Dzieci

Aplikacja nie jest projektowana w celu zbierania danych osobowych od dzieci.

Zmiany

Ta polityka powinna być aktualizowana, jeśli aplikacja zacznie korzystać z nowych źródeł danych, analityki, raportowania awarii, kont użytkowników, płatności lub innych zewnętrznych dostawców.

Privacy Policy

About the app

Navi Live is a walking navigation app designed with blind and low-vision users in mind.

What the app uses

  • device location, to search nearby places, build walking routes, and support active navigation,
  • local device storage, to save favorites, settings, tutorial state, and recent route state,
  • network access, to send search, reverse geocoding, and routing requests to public OpenStreetMap-based services used by the app.

What the app does not currently use

  • user accounts,
  • advertising SDKs,
  • analytics SDKs,
  • crash reporting SDKs,
  • in-app purchases,
  • payment processing,
  • in-app messaging or chat.

How location is used

Location is used only to:

  • determine the current position,
  • search for nearby places,
  • build and recalculate routes,
  • support live navigation guidance.

Local data

The app may store some information locally on the device, such as:

  • saved favorites,
  • app settings,
  • tutorial and onboarding state,
  • last route state.

This information is used only to improve the app experience on the same device.

Third-party services

The app sends necessary requests to public OpenStreetMap-based search and routing services in order to perform core navigation functions.

These requests may include:

  • destination search text,
  • approximate route endpoints,
  • coordinates needed for route calculation or reverse geocoding.

Data sharing

The app does not currently maintain a developer-operated user account system or a developer-operated cloud profile tied to your identity.

Children

The app is not designed to collect personal information from children.

Changes

This policy should be updated whenever the app starts using new data sources, analytics, crash reporting, accounts, payments, or different third-party providers.
